# Sample Assets

This directory contains a small subset of images from the Kodak Lossless True Color Image Suite:

- `kodim04.png`
- `kodim07.png`
- `kodim08.png`
- `kodim23.png`

Source:

- Richard W. Franzen, "Kodak Lossless True Color Image Suite"
- [https://r0k.us/graphics/kodak/](https://r0k.us/graphics/kodak/)

The source page states that it is the author's understanding that these images were released by Eastman Kodak Company for unrestricted usage. This repository includes only a few files for educational testing and debug examples.

If redistribution requirements for these files matter for your use case, obtain the dataset directly from the source above.
